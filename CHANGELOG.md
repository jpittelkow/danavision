# Changelog

All notable changes to DanaVision will be documented in this file.

## [Unreleased]

### Added
- Build versioning system with VERSION file as single source of truth
- Version display in application sidebar
- CHANGELOG.md for tracking releases
- Version bump script for easy version management

## [1.0.0] - 2026-01-24

### Added
- Initial release of DanaVision
- Smart shopping list management with price tracking
- AI-powered product identification and price search
- Multi-vendor price comparison
- Shop Local feature for nearby store discovery
- Smart Add feature for quick product entry
- Dashboard with price trends and insights
- Email notifications for price drops
- Dark/light theme support
- Mobile-responsive design

### Infrastructure
- Docker containerized deployment
- GitHub Actions CI/CD pipeline
- Playwright E2E testing
- Pest PHP unit testing
